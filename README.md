# Automated Employee Onboarding and Offboarding System

## Project Overview
This project automates the Employee Onboarding and Offboarding process using ServiceNow. It reduces manual work by automating employee requests, approvals, record creation, and lifecycle management.

## Features
- Employee Onboarding Request
- Employee Offboarding Request
- Service Catalog Form
- Flow Designer Automation
- Employee Lifecycle Table
- Access Control (ACL)
- Automated Data Handling
- QA Testing
- Data Integrity Validation

## Technologies Used
- ServiceNow
- Flow Designer
- Service Catalog
- Custom Tables
- Access Control Lists (ACL)
- Workflow Automation

## Project Workflow
1. User submits an onboarding/offboarding request.
2. Service Catalog captures employee details.
3. Flow Designer processes the request.
4. Employee Lifecycle record is created automatically.
5. Access control secures the data.
6. QA testing verifies successful execution.

## Project Status
Completed successfully as part of the ServiceNow System Administrator project.
